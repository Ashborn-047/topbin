# topbin Agent Rules & Guidelines

This repository is dedicated to building the ultimate, fully open, public-domain historical football database. 

## 🚨 Core Directive: Historical Depth First
When contributing to this database, **do not focus only on current/recent seasons**. We aim to capture as much historical data as possible going back several decades (and over a century for international data and established leagues).

### Data Scope
1. **International Tournaments**: Keep full match-by-match history from the inception of the tournament (e.g., World Cups from 1930 to present).
2. **Clubs & Leagues**: Go back as many decades as data exists (e.g., English league results back to the late 19th/early 20th century).
3. **Rosters**: Aim to build squads/rosters for historic seasons (e.g., legendary World Cup squads, historic treble-winning club squads).
4. **Data Structuring**:
   - Store historical archive data in structured subfolders (e.g., `data/competitions/world-cups/archive/`, `data/leagues/archive/`).
   - Standardize schemas across all historical eras so that queries can seamlessly span decades.
